Enum Error

    Invalid,
    Unsupported,
    LengthExceedsMax,
    LengthMismatch,

    InvalidHex,
    Io(Error),
    DepthLimitExceeded,
    LengthLimitExceeded,
}
Expand description

Error contains all errors returned by functions in this crate. It can be compared via PartialEq, however any contained IO errors will only be compared on their ErrorKind.
